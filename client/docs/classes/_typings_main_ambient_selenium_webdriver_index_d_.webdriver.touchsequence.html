Class TouchSequence

Class for defining sequences of user touch interactions. Each sequence will not be executed until {@link #perform} is called.

Example:

new webdriver.TouchSequence(driver).
    tapAndHold({x: 0, y: 0}).
    move({x: 3, y: 4}).
    release({x: 10, y: 10}).
    perform();
