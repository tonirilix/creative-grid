chord

chord: function
  • Defined in typings/main/ambient/selenium-webdriver/index.d.ts:2207

Simulate pressing many keys at once in a "chord". Takes a sequence of webdriver.Keys or strings, appends each of the values to a string, and adds the chord termination key ({@link webdriver.Key.NULL}) and returns the resultant string.

Note: when the low-level webdriver key handlers see Keys.NULL, active modifier keys (CTRL/ALT/SHIFT/etc) release via a keyup event.

param

The key sequence to concatenate.

returns

The null-terminated key sequence.

see

http://code.google.com/p/webdriver/issues/detail?id=79
