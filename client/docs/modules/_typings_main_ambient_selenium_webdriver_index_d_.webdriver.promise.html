consume

  • consume<T>(generatorFn: Function, opt_self?: any, ...var_args: any[]): Promise<T>
    • Defined in typings/main/ambient/selenium-webdriver/index.d.ts:1098

    Consumes a {@code GeneratorFunction}. Each time the generator yields a promise, this function will wait for it to be fulfilled before feeding the fulfilled value back into {@code next}. Likewise, if a yielded promise is rejected, the rejection error will be passed to {@code throw}.

    Example 1: the Fibonacci Sequence.

    
webdriver.promise.consume(function* fibonacci() {
  var n1 = 1, n2 = 1;
  for (var i = 0; i < 4; ++i) {
    var tmp = yield n1 + n2;
    n1 = n2;
    n2 = tmp;
  }
  return n1 + n2;
}).then(function(result) {
  console.log(result);  // 13
});

    Example 2: a generator that throws.

    
webdriver.promise.consume(function* () {
  yield webdriver.promise.delayed(250).then(function() {
    throw Error('boom');
  });
}).thenCatch(function(e) {
  console.log(e.toString());  // Error: boom
});
    throws

    {TypeError} If the given function is not a generator.

    Type parameters

    Parameters

    • generatorFn: Function

      The generator function to execute.

    • Optional opt_self: any

      The object to use as "this" when invoking the initial generator.

    • Rest ...var_args: any[]

      Any arguments to pass to the initial generator.

    Returns Promise<T>

    A promise that will resolve to the generator's final result.

fullyResolved

  • fullyResolved<T>(value: any): Promise<T>
    • Defined in typings/main/ambient/selenium-webdriver/index.d.ts:1133

    Returns a promise that will be resolved with the input value in a fully-resolved state. If the value is an array, each element will be fully resolved. Likewise, if the value is an object, all keys will be fully resolved. In both cases, all nested arrays and objects will also be fully resolved. All fields are resolved in place; the returned promise will resolve on {@code value} and not a copy.

    Warning: This function makes no checks against objects that contain cyclical references:

    var value = {}; value['self'] = value; webdriver.promise.fullyResolved(value); // Stack overflow.

    Type parameters

    • T

    Parameters

    • value: any

      The value to fully resolve.

    Returns Promise<T>

    A promise for a fully resolved version of the input value.
